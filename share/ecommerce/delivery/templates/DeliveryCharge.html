In the Countries text area enter lines containing an iso country code and a delivery rate code separated by a ';', for example:

    250;zone1
    826;express
    826;standard

A list of iso country codes can be found here. The delivery rate code is defined in the Rates text area.

When the form is submitted the country matching the country code is added as a prefix to the line.

In the Rates text area enter lines containing a code (this must be unique), a label and a price separated by a ';', for example:

    standard;standard;5.00
    express;express;10.00
    zone1;standard;10.00
